# 在 LobeHub 中使用文心千帆

<Image cover src={'/blog/assets89b0698da3476c6df24ba1f0a07e438e.webp'} />

[文心千帆](https://qianfan.cloud.baidu.com/)是百度推出的一个人工智能大语言模型平台，支持多种应用场景，包括文学创作、商业文案生成、数理逻辑推算等。该平台具备跨模态、跨语言的深度语义理解与生成能力，广泛应用于搜索问答、内容创作和智能办公等领域。

本文将指导你如何在 LobeHub 中使用文心千帆。

<Steps>
  ### 步骤一：获得文心千帆的 API Key

  - 注册并登录 [百度智能云控制台](https://console.bce.baidu.com/)
  - 进入 `百度智能云千帆 ModelBuilder`
  - 在左侧菜单中选择 `API Key`

  <Image alt={'API Key'} inStep src={'/blog/assets67a17a6c66592ad85dbcf190e73b182d.webp'} />

  - 点击创建 API Key
  - 在 `服务` 中选择 `千帆ModelBuilder`
  - 在 `资源` 中选择 `所有资源`
  - 点击 `确定` 按钮
  - 复制 `API Key` 并妥善保存

  <Image alt={'创建密钥'} inStep src={'/blog/assets87c281587b15f05b6b4e1afcd5bb47e8.webp'} />

  <Image alt={'复制密钥'} inStep src={'/blog/assets257e1a3a0d99ee043a4c6cb90e160a2b.webp'} />

  ### 步骤二：在 LobeHub 中配置文心千帆

  - 访问 LobeHub 的`设置`界面
  - 在`AI 服务商`下找到 `文心千帆` 的设置项

  <Image alt={'填入 API 密钥'} inStep src={'/blog/assets8f95f09ce51ad5917107d84db1e980ab.webp'} />

  - 填入获得的 `API Key`
  - 为你的 AI 助手选择一个文心千帆的模型即可开始对话

  <Image alt={'选择文心千帆模型并开始对话'} inStep src={'/blog/assets768ec6fd300785186b202437985857c4.webp'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考文心千帆的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeHub 中使用文心千帆提供的模型进行对话了。

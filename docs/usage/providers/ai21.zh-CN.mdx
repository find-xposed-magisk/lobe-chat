# 在 LobeHub 中使用 AI21 Labs

<Image cover src={'/blog/assetsa25c48c9faa225bf6f72658e5bd58d64.webp'} />

[AI21 Labs](https://www.ai21.com/) 是一家专注于人工智能的公司，提供先进的语言模型和 API 服务，旨在帮助开发者和企业利用自然语言处理技术。其旗舰产品 "Jamba" 系列模型能够进行复杂的语言理解和生成任务，广泛应用于内容创作、对话系统等领域。

本文将指导你如何在 LobeHub 中使用 AI21 Labs。

<Steps>
  ### 步骤一：获得 AI21 Labs 的 API Key

  - 注册并登录 [AI21 Studio](https://studio.ai21.com)
  - 点击 `用户头像` 菜单，点击 `API Key`
  - 复制并保存生成的 API 密钥

  <Image alt={'复制 API 密钥'} inStep src={'/blog/assets0f847842a5dedf7bef1f534278aec584.webp'} />

  ### 步骤二：在 LobeHub 中配置 AI21 Labs

  - 访问 LobeHub 的`设置`界面
  - 在`AI 服务商`下找到 `AI21labs` 的设置项

  <Image alt={'填入 API 密钥'} inStep src={'/blog/assetsc16177645281b332883403e7f193f6e3.webp'} />

  - 填入获得的 API 密钥
  - 为你的 AI 助手选择一个 AI21 Labs 的模型即可开始对话

  <Image alt={'选择 AI21 Labs 模型并开始对话'} inStep src={'/blog/assets5f1a6cb003752055b9ed131c1715154c.webp'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考 AI21 Labs 的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeHub 中使用 AI21 Labs 提供的模型进行对话了。

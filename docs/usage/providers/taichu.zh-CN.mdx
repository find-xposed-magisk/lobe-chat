# 在 LobeHub 中使用紫东太初

<Image alt={'在 LobeHub 中使用太初'} cover src={'/blog/assetsd3fefc9a525701b9d0f25116cea2ff00.webp'} />

本文将指导你如何在 LobeHub 中使用紫东太初：

<Steps>
  ### 步骤一：获取紫东太初 API 密钥

  - 创建一个[紫东太初](https://ai-maas.wair.ac.cn/)账户
  - 创建并获取 [API 密钥](https://ai-maas.wair.ac.cn/#/settlement/api/key)

  <Image alt={'创建 API Key'} inStep src={'/blog/assets04ab03ac7920031925f7ee27846b3f7d.webp'} />

  ### 步骤二：在 LobeHub 中配置紫东太初

  - 访问 LobeHub 的`设置`界面
  - 在`AI 服务商`下找到`紫东太初`的设置项

  <Image alt={'填入 API 密钥'} inStep src={'/blog/assets8f3458d794828c38220e88b66e994e2a.webp'} />

  - 填入获得的 API 密钥
  - 为你的 AI 助手选择一个紫东太初的模型即可开始对话

  <Image alt={'选择太初模型并开始对话'} inStep src={'/blog/assets94499977be2f01c795b9876e4fe60709.webp'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考紫东太初的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeHub 中使用紫东太初提供的模型进行对话了。

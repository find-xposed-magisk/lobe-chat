# 在 LobeHub 中使用 Novita AI

<Image alt={'在 LobeHub 中使用 Novita AI'} cover src={'/blog/assets85af5a2a51b851fe125055d374cc8263.webp'} />

[Novita AI](https://novita.ai/) 是一个 AI API 平台，它提供多种大语言模型与 AI 图像生成的 API 服务。支持 Llama3 (8B, 70B)，Mistral 和其他最新的模型。

本文档将指导你如何在 LobeHub 中使用 Novita AI:

<Steps>
  ### 步骤一：注册 Novita AI 账号并登录

  - 访问 [Novita.ai](https://novita.ai/) 并创建账号
  - 你可以使用 Google 或者 Github 账号登录
  - 注册后，Novita AI 会赠送 0.5 美元的使用额度

  <Image alt={'注册 Novita AI'} height={457} inStep src={'/blog/assets86924c724c66931cf61417dbdcc04ee8.webp'} />

  ### 步骤二：创建 API 密钥

  - 访问 Novita AI 的 [密钥管理页面](https://novita.ai/dashboard/key) ，创建并且复制一个 API 密钥.

  <Image alt={'创建 Novita AI API 密钥'} inStep src={'/blog/assets6f2a84bee4245ca507e98e96247d5c5e.webp'} />

  ### 步骤三：在 LobeHub 中配置 Novita AI

  - 访问 LobeHub 的 `设置` 界面
  - 在 `AI 服务商` 下找到 `novita.ai` 的设置项
  - 打开 novita.ai 并填入获得的 API 密钥

  <Image alt={'在 LobeHub 中输入 Novita AI API 密钥'} inStep src={'/blog/assets05d786345e99f92d11baae5667a04a62.webp'} />

  - 为你的助手选择一个 Novita AI 模型即可开始对话

  <Image alt={'Select and use Novita AI model'} inStep src={'/blog/assets9db16311eb6772ea74eb63dd2d397bc0.webp'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考 Novita AI 的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeHub 中使用 Novita AI 提供的模型进行对话了。

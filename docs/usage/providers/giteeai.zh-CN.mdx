# 在 LobeHub 中使用 Gitee AI

<Image cover src={'/blog/assetsd94f3e0cf32639bea46dbf92e0862f89.webp'} />

[Gitee AI](https://ai.gitee.com/) 是一个基于 Git 代码托管技术的开源平台，专为人工智能（AI）应用场景设计。它旨在为开发者和企业提供一站式的 AI 应用开发服务，包括模型体验、推理、微调和部署等功能。

本文将指导你如何在 LobeHub 中使用 Gitee AI。

<Steps>
  ### 步骤一：获取 Gitee AI 的 API 密钥

  - 注册并登录 [Gitee AI 官网](https://ai.gitee.com/)
  - 在工作台中购买并充值 `Serverless API`

  <Image alt={'Gitee Serverless API'} inStep src={'/blog/assets5a2f360c19fcf9a037b2d1609479b713.webp'} />

  - 在 `设置` 中点击 `访问令牌` 界面
  - 创建一个新的访问令牌
  - 在弹出窗口中保存访问令牌

  <Image alt={'Gitee Serverless API'} inStep src={'/blog/assetsf1e1ca1adaac36881ec6c3b2ce1a099e.webp'} />

  <Callout type={'warning'}>
    妥善保存弹窗中的访问令牌，它只会出现一次，如果不小心丢失了，你需要重新创建一个访问令牌。
  </Callout>

  ### 步骤二：在 LobeHub 中配置 Gitee AI

  - 访问 LobeHub 的`设置`界面
  - 在`AI 服务商`下找到 `Gitee AI` 的设置项

  <Image alt={'填入 API 密钥'} inStep src={'/blog/assetsf5a62c963127764ebdf1cd226fac3dac.webp'} />

  - 填入获得的 API 密钥
  - 为你的 AI 助手选择一个 Gitee AI 的模型即可开始对话

  <Image alt={'选择 Gitee AI 模型并开始对话'} inStep src={'/blog/assetsec0f694c9f6140620217bde441440170.webp'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考 Gitee AI 的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeHub 中使用 Gitee AI 提供的模型进行对话了。

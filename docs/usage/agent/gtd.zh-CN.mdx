# 任务工具

任务工具（`lobe-task`）是内置技能，能让助理成为你的任务管理伙伴。它默认为每个助理启用，无需手动开启。你可以捕捉想法、追踪进度、把工作拆分为子任务、将子任务分配给其他助理，并为需要反复执行的工作配置自动化 —— 这一切都通过自然语言对话完成。突如其来的灵感、待办的家务，或是一份多步骤的项目计划 —— 助理都能记录、梳理，并推动它落地。

## 任务工具默认已开启

与市场技能不同，`lobe-task` 是每个助理默认拥有的工具之一 —— 你无需前往助理档案页面点击 \*\*+ 添加技能 \*\* 来开启它。如果你自定义过某个助理的工具列表并移除了它，可以在助理档案 → **+ 添加技能** → **任务工具** 中重新启用；但对于新建的助理，它已经处于启用状态。

## 创建任务

在会话中直接发送你的计划 —— 助理会自动识别意图并创建任务。无需填表，无需额外步骤。

**示例：**

- 「把「审阅 Q1 报告」加入我明天的待办」
- 「这周需要给牙医打个电话」
- 「提醒我周五前提交方案」
- 「把这个拆成子任务：设计稿、收集反馈、定稿」
- 「把 SEO 审查这个子任务分配给我的营销助理」

## 完成和更新任务

通过对话指令更新任务，或者让某次运行结束后自动更新自己的状态。助理会处理好这些细节。

**示例：**

- 「把「审阅 Q1 报告」标为已完成」
- 「今天我的待办有哪些？」
- 「把牙医电话改到下周一」
- 「给我看看所有未完成的任务」
- 「删掉方案那个任务，我已经交了」

## 子任务与分配

任务支持子任务和依赖关系，每个任务还可以指定一个执行助理：

- **子任务** — 将一个大任务拆分为有序的多个部分（例如第一章、第二章、第三章），它们会嵌套显示在父任务下。
- **依赖关系** — 将某个任务标记为依赖另一个任务，只有在其依赖的任务完成后才会开始。
- **分配** — 将任务分配给指定的助理，该任务被触发时会由这个助理执行。
- **验收校验** — 开启验收校验后，会有一个独立的审查者按照设定的验收要求检查产出，通过后任务才算完成。

## 自动化任务

任务不仅可以是一次性的，也可以配置为周期性执行的自动化工作 —— 设置 cron 计划或固定的心跳间隔后，助理会持续自动运行该任务。完整流程参见[定时任务](/zh/docs/usage/agent/scheduled-task)：在助理会话中打开左侧的**任务**面板，选中某个任务，在其详情的**自动化**区域配置计划。

## 使用 CLI

如果你使用 LobeHub CLI，同一套任务系统也可以通过命令行的 `lh task` 使用：

```
lh task list                                   # 列出你的任务
lh task view T-1                               # 查看某个任务的详情
lh task create -i "撰写发布说明"                 # 创建任务
lh task edit T-1 --status completed            # 更新任务状态
```

这对于用脚本创建任务、在自动化流程中检查状态，或者在编码会话中一并管理任务而不用切换到界面非常有用。

## 使用场景

**随时捕捉** — 会议中突然想到一件事？用一句话告诉助理，无需切换应用或打开任务管理器。

**周计划** — 开始一段对话：「这周我需要做这些……」助理会帮你整理成可执行的任务和子任务列表。

**切换上下文** — 从一个项目转到另一个？问「营销项目上我还有什么没做完？」就能接着上次的进度继续。

**分派工作** — 把一个项目拆分为多个子任务，分配给最合适的助理，再让依赖关系和自动化计划驱动后续执行。

**协作准备** — 团队同步前，问「上周我有哪些待办事项？」带着准备去开会。

## 任务工具与其他功能

| 功能                                          | 最适合                             |
| ------------------------------------------- | ------------------------------- |
| **任务工具**                                    | 任务列表、子任务、截止日期、分配、进度追踪 —— 通过对话管理 |
| [笔记本](/zh/docs/usage/agent/notebook)        | 保存笔记、报告、研究资料 —— 与话题关联的文档        |
| [定时任务](/zh/docs/usage/agent/scheduled-task) | 自动化周期执行 —— 由任务的自动化计划驱动          |

日常任务管理和分派用任务工具；需要任务按计划自动执行时，配置该任务的自动化。

## 使用技巧

- **明确时间** — 「明天上午 9 点」或「本周内」有助于助理在合适时机提醒你。
- **定期回顾** — 每周初问「这周有什么要交的？」保持对承诺的掌控。
- **与笔记本配合** — 把会议纪要保存到笔记本，再让助理从中提取待办事项到任务中。
- **多步骤工作用子任务** — 把工作拆成带依赖关系的子任务，能让大项目保持有序，也便于独立分派各个部分。

# Deploy LobeHub with Netlify

If you want to deploy LobeHub on Netlify, you can follow these steps:

## Deploy LobeHub with Netlify

<Steps>
  ### Fork the LobeHub Repository

  Click the Fork button to fork the LobeHub repository to your GitHub account.

  ### Import to Netlify Workspace

  <Callout>
    After testing, it is currently not supported to have a one-click deployment button similar to
    Vercel/Zeabur. The reason is unknown. Therefore, manual import is required.
  </Callout>

  Click "Import from git"

  <Image alt={'Click "Import from git" in the Netlify workspace'} height={362} inStep src={'/blog/assets28616219/31b999e5-2a74-45fc-935b-f036e72a684d.webp'} />

  Then click "Deploy with Github" and authorize Netlify to access your GitHub account.

  <Image alt={'Authorize Netlify to access your GitHub account'} height={273} inStep src={'/blog/assets28616219/c9c58141-5ec6-43f1-8d97-0a84a04dcdba.webp'} />

  Next, select the LobeHub project:

  <Image alt={'Select the LobeHub github project'} height={228} inStep src={'/blog/assets28616219/6c3968a8-fbbb-4268-a587-edaced2d96af.webp'} />

  ### Configure Site Name and Environment Variables

  In this step, you need to configure your site, including the site name, build command, and publish directory. Fill in your site name in the "Site Name" field. If there are no special requirements, you do not need to modify the remaining configurations as we have already set the default configurations.

  <Image alt={'Configure LobeHub site name'} height={712} inStep src={'/blog/assets28616219/f3885537-6d43-422f-b1b8-e70732401025.webp'} />

  Click the "Add environment variables" button to add site environment variables if needed:

  <Image alt={'Add LobeHub site environment variables'} height={537} inStep src={'/blog/assets28616219/3b607482-4d99-455a-bc10-3090dd4fe3c5.webp'} />

  <Callout type={'tip'}>
    For a complete list of environment variables supported by LobeHub, please refer to the [ðŸ“˜
    Environment Variables](/docs/self-hosting/environment-variables)
  </Callout>

  Finally click "Deploy lobe-chat" to enter the deployment phase

  <Image alt={'Environment variables added'} height={600} inStep src={'/blog/assets28616219/c9f74ec4-ce63-4ce9-b9e2-34bda6fda10b.webp'} />

  ### Wait for Deployment to Complete

  After clicking deploy, you will enter the site details page, where you can click the "Deploying your site" in blue or the "Building" in yellow to view the deployment progress.

  <Image alt={'Netlify site details page'} height={396} inStep src={'/blog/assets28616219/a7fd85d5-fd32-4756-814e-ff7ab7567fe1.webp'} />

  Upon entering the deployment details, you will see the following interface, indicating that your LobeHub is currently being deployed. Simply wait for the deployment to complete.

  <Image alt={'LobeHub deployment in progress'} height={325} inStep src={'/blog/assets28616219/1ed8b13d-046e-47c8-bd61-116ffdf5d01b.webp'} />

  During the deployment and build process:

  <Image alt={'Deployment in progress'} height={558} inStep src={'/blog/assets28616219/1c82d707-cb6f-4924-b246-a5235a919864.webp'} />

  ### Deployment Successful, Start Using

  If your Deploy Log in the interface looks like the following, it means your LobeHub has been successfully deployed.

  <Image alt={'Deployment Successful'} height={558} inStep src={'/blog/assets28616219/c1f945d1-f3e2-4100-b6bb-24d4cb13c438.webp'} />

  At this point, you can click on "Open production deploy" to access your LobeHub site.

  <Image alt={'Access Your LobeHub Site'} height={527} inStep src={'/blog/assets28616219/b04723eb-64ad-4028-a901-dc4e4ee2d0c1.webp'} />
</Steps>

## Set up Custom Domain (Optional)

You can use the subdomain provided by Netlify, or choose to bind a custom domain. Currently, the domain provided by Netlify has not been contaminated, and can be accessed directly in most regions.

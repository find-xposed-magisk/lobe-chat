# 更清爽的对话与拖拽配图

## 新功能

- **已完成回合自动折叠。** 较长的助理运行会把已完成的回合折叠为简要摘要，只保留最新一条展开，长对话更易阅读。
- **拖拽上传参考图。** 把图片直接拖入输入框即可作为参考图，无需打开上传弹窗。
- **审批集中提醒。** 全局审批提醒会汇总陆续到来的人工审批请求；你现在可以选择「不再询问」，避免同一操作反复弹窗。
- **桌面端完成通知。** 助理完成后，桌面端会发送一条通知并直接链接到结果，点击即可唤回最小化的窗口。
- **更聪明的构建建议。** 助理与群组构建器会根据你正在进行的工作给出建议，并随你的反馈持续调整。
- **移动端支持技能与插件。** 现在可以在移动端访问助理技能与插件。

## 体验优化

- 新增 Doubao-Seed-2.1 系列模型。
- Amazon Bedrock 支持 API Key 认证。
- AiHubMix 支持自定义或备用 Base URL。
- Kimi 模型对思考内容的保留更稳定。
- 在桌面端，Claude Code 运行遇到过载错误时会自动重试，并就地替换结果。
- 任务完成后会把结果交回创建它的对话，并把任务标识链接到任务详情页；任务回调改为在侧边面板打开，而不再跳转离开。
- 当文本后紧跟工具调用时，流式文本会跳过动画，回复更迅速。

## 问题修复

- 正确处理 GLM-5.2 的参数与模型描述。
- 模型不可用时，对话会给出明确提示。
- DeepSeek V4 的思考内容不再混入推理通道。
- 聊天滚动位置仅在五分钟内恢复，重新打开旧话题会回到底部。
- 跨助理调用失败时会说明真实原因，而不是悄无声息地失败。
- 打开已被删除的负责助理的任务时不再卡死。
- 可以重新在设置中断开 OAuth 连接器。
- 内置工具卡片会显示真实的任务与助理数据。
